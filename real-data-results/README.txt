The files weighted.averages.real.data.rds contains the averaged statistics such as the overall genotype error rate and overall deletion rate for a set of MAF intervals, calculated on the UK Biobank data. weighted.averages.real.data.boot.rds contains these statistics calculated on the bootstrapped observed trio genotypes. 

est.res.pt.rds contains the point estimates from all MAF intervals in the UK Biobank sequence data.

The files est.res.13.*.rds contain the result of applying the model to each set of bootstrapped observed trio genotypes. 

The files resamp.trios.13.*.rds contain the bootstrapped trio ids.
